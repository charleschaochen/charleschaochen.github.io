      在tomcat服务器上发布J2EE应用后，一般我们通过http://localhost:8080/来访问。

      String path = request.getContextPath();
    String basePath = request.getScheme() + "://"
            + request.getServerName() + ":" + request.getServerPort()
            + path + "/";
